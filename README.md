# Database Systems Open-Ended Lab

## Overview

This repository contains open-ended laboratory exercises and projects focused on **Database Systems**, with a particular emphasis on **Vector Databases**. These labs are designed to provide hands-on experience with modern database technologies and concepts.

## 📚 Contents

### Lab Exercises

- **Lab 14: Vector Database** (`Lab_14_Vector_Database.ipynb`)
  - Comprehensive exploration of vector databases and their applications
  - Practical implementation examples
  - Use cases and best practices for vector-based data storage and retrieval

### Supporting Files

- `metadata.csv` - Metadata and configuration files for lab exercises
- `README.md` - This documentation file

## 🎯 Learning Objectives

This lab series aims to help you:

- Understand core database systems concepts and architecture
- Explore vector databases and their unique capabilities
- Implement database operations and queries
- Work with real-world database scenarios
- Develop practical skills in modern database technologies

## 🚀 Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook or JupyterLab
- Required Python packages (specified in individual notebooks)

### Running the Labs

1. Clone this repository:
   ```bash
   git clone https://github.com/AbdulAzeemHashmi/Database-Systems-Open-Ended-Lab.git
   cd Database-Systems-Open-Ended-Lab
   ```

2. Install required dependencies:
   ```bash
   pip install -r requirements.txt  # if available
   # or install packages as needed
   ```

3. Open the notebook in Jupyter:
   ```bash
   jupyter notebook Lab_14_Vector_Database.ipynb
   ```

4. Follow the exercises and instructions within the notebook
